Date Difference

This tab allows you to calculate the difference between two dates. You can choose to calculate the difference in days, weeks, months, or years. You can also choose to include or exclude the start date in the calculation.

Enter the start date and end date in the input fields and select the units you want to calculate the difference in. Click the "Calculate" button to see the result.

For example, if you enter the start date as 01/01/2020 and the end date as 01/01/2021, and select "Years" as the unit, the result will be 1 year.
